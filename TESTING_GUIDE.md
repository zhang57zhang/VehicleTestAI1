# VehicleTestAI1 - E2E 测试指南

## 🚀 快速启动

### 1. 配置AI服务
```bash
# 在 backend/.env 文件中配置GLM-4.7 API
echo "GLM_API_KEY=your_api_key_here" > backend/.env
echo "GLM_MODEL=glm-4.7" >> backend/.env
```

### 2. 启动后端服务
```bash
cd /home/qw/.openclaw/workspace/VehicleTestAI1/backend
python3 app.py
```

### 3. 启动前端服务（新终端）
```bash
cd /home/qw/.openclaw/workspace/VehicleTestAI1
python3 -m http.server 8080
```

### 4. 访问应用
打开浏览器访问: http://localhost:8080
