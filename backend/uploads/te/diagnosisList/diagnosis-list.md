## 1. DTC编码规范

### 1.1 编码格式
采用ISO 15031-6 / ISO 14229-1标准DTC格式：

```
┌─────────────────────────────────────────┐
│  DTC Code (3 Bytes)                      │
│  ┌────┬────┬────┬────┬────┬────┬────┐   │
│  │ P0 │ P1 │ P2 │ P3 │ P4 │ P5 │ P6 │   │
│  │       │    │    │    │    │    │   │
│  │前2位   │  子系统 │  具体故障码    │   │
│  └────┴────┴────┴────┴────┴────┴────┘   │
└─────────────────────────────────────────┘

P0P1: 故障类型标识
P2P3: 子系统标识
P4P5P6: 具体故障序号
```

### 1.2 故障类型
| 前缀 | 类型 |
|------|------|
| P0 | 动力总成 |
| P1 | 电机控制 |
| P2 | 电池系统 |
| C0 | 底盘系统 |
| U0 | 网络通信 |
| B0 | 车身系统 |
