# 车载电驱动MCU需求文档

**文档编号**：REQ-EV-MCU-2026-001  
**版本**：V1.0  
**日期**：2026-04-07  
**作者**：电驱动系统团队

---

## 1. 概述

### 1.1 项目背景
本文档定义车载电驱动电机控制器（MCU）的功能需求，用于指导系统设计、开发和测试工作。

### 1.2 适用范围
适用于纯电动汽车及混合动力汽车的高压电驱动系统MCU。

### 1.3 术语定义
| 术语 | 定义 |
|------|------|
| MCU | Motor Control Unit，电机控制器 |
| VCU | Vehicle Control Unit，整车控制器 |
| BMS | Battery Management System，电池管理系统 |
| FOC | Field Oriented Control，磁场定向控制 |
| DTC | Diagnostic Trouble Code，诊断故障码 |

## 4. 接口需求

### 4.1 外部接口
| 接口类型 | 数量 | 说明 |
|----------|------|------|
| CAN总线 | 2路 | CAN1:整车网关, CAN2:诊断 |
| 高压接口 | 2路 | 直流母线输入, 电机输出 |
| 低压接口 | 1路 | 12V供电和IO信号 |
| 传感器接口 | 4路 | 旋变、温度、电流传感器 |

### 4.2 信号接口
| 信号名称 | 类型 | 说明 |
|----------|------|------|
| ENABLE | 输入 | 使能信号 |
| FAULT | 输出 | 故障信号 |
| HVIL | 输入/输出 | 高压互锁 |
| VSP | 输出 | 车速脉冲 |
