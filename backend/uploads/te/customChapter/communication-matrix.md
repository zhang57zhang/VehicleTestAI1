## 2. CAN ID分配

### 2.1 ID规划原则
| ID范围 | 用途 | 优先级 |
|--------|------|--------|
| 0x000-0x0FF | 保留 | - |
| 0x100-0x1FF | VCU→MCU 控制指令 | 高 |
| 0x200-0x2FF | MCU→VCU 状态反馈 | 高 |
| 0x300-0x3FF | MCU→VCU 故障诊断 | 最高 |
| 0x400-0x4FF | BMS报文 | 中 |
| 0x500-0x5FF | 自动化测试专用 | 低 |
