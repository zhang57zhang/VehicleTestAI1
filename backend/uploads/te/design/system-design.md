## 1. 系统架构

### 1.1 系统框图

```
                    ┌─────────────────────────────────────────┐
                    │              整车网络                    │
                    └──────────────┬──────────────────────────┘
                                   │ CAN Bus
                    ┌──────────────▼──────────────────────────┐
                    │              MCU 控制器                   │
                    │  ┌─────────────────────────────────┐    │
                    │  │        应用层 (Application)       │    │
                    │  │  ┌─────────┐ ┌─────────┐ ┌─────┐ │    │
                    │  │  │扭矩控制 │ │转速控制 │ │诊断 │ │    │
                    │  │  └────┬────┘ └────┬────┘ └──┬──┘ │    │
                    │  └───────┼──────────┼─────────┼────┘    │
                    │          └──────────┼─────────┘         │
                    │  ┌──────────────────▼───────────────┐   │
                    │  │        FOC 控制层                  │   │
                    │  │   ┌───────┐  ┌───────┐  ┌─────┐  │   │
                    │  │   │  Park │  │  Clarke │ │ PID │  │   │
                    │  │   └───────┘  └───────┘  └─────┘  │   │
                    │  └──────────────────┬───────────────┘   │
                    │  ┌──────────────────▼───────────────┐   │
                    │  │        功率驱动层                  │   │
                    │  │     IGBT 逆变器 / PWM 生成         │   │
                    │  └──────────────────┬───────────────┘   │
                    └─────────────────────┼───────────────────┘
                                          │ 3-Phase
                              ┌───────────▼───────────┐
                              │     PMSM 电机         │
                              └───────────────────────┘
```

### 1.2 模块划分

| 模块 | 功能 | 说明 |
|------|------|------|
| 通信管理模块 | CAN/诊断通信 | 接收指令、发送状态 |
| 扭矩控制模块 | 扭矩闭环控制 | 响应VCU扭矩请求 |
| 转速控制模块 | 转速闭环控制 | 转速调节 |
| FOC控制模块 | 磁场定向控制 | Id/Iq电流控制 |
| PWM生成模块 | 逆变器驱动 | SVPWM调制 |
| 保护模块 | 故障保护 | 过流/过温/过压保护 |
| 诊断模块 | 故障诊断 | DTC管理 |

## 3. 软件架构

### 3.1 软件分层

```
┌─────────────────────────────────────────────────┐
│                  应用层 (APP)                    │
│  扭矩控制 | 转速控制 | 诊断管理 | 状态管理        │
├─────────────────────────────────────────────────┤
│                  服务层 (Services)               │
│  CAN服务 | 诊断服务 | NVM服务 | 看门狗服务        │
├─────────────────────────────────────────────────┤
│                  驱动层 (Drivers)                │
│  ADC驱动 | PWM驱动 | CAN驱动 | SPI驱动 | GPIO   │
├─────────────────────────────────────────────────┤
│                  HAL层                           │
│  寄存器访问 | 中断管理 | DMA管理                  │
└─────────────────────────────────────────────────┘
```

### 3.2 任务调度

| 任务 | 周期 | 优先级 | 说明 |
|------|------|--------|------|
| 电流环 | 100μs | 最高 | FOC电流控制 |
| 速度环 | 1ms | 高 | 转速控制 |
| 扭矩控制 | 10ms | 中 | 扭矩管理 |
| CAN发送 | 10ms | 中 | 状态上报 |
| CAN接收 | 事件 | 中 | 指令接收 |
| 诊断处理 | 50ms | 低 | UDS诊断 |
| 状态监控 | 100ms | 低 | 温度/电压监控 |

## 4. 控制策略

### 4.1 扭矩控制策略

```
扭矩请求 ──┬──> 扭矩斜率限制 ──> 扭矩/电流转换
           │
           └──> 功率限制 ──> 最大扭矩限制
                                │
                                ▼
                          Id/Iq电流指令
                                │
                                ▼
                     ┌─────────────────────┐
                     │   FOC 控制器        │
                     │  ┌───────────────┐  │
                     │  │  电流环PID    │  │
                     │  │  Va, Vb, Vc   │  │
                     │  └───────────────┘  │
                     └─────────────────────┘
                                │
                                ▼
                          SVPWM调制
                                │
                                ▼
                          PWM占空比
```

### 4.2 FOC控制框图

```
     θ (电机角度)

        ▼
┌───────────────────────────────────────────────┐
```

### 4.3 保护策略

| 保护类型 | 阈值 | 动作 |
|----------|------|------|
| 过流I级 | 500A | 降功率 |
| 过流II级 | 600A | 切断输出 |
| IGBT过温 | 125℃ | 降功率 |
| IGBT极限 | 150℃ | 切断输出 |
| 电机过温 | 140℃ | 降功率 |
| 高压过压 | 450V | 切断输出 |
| 高压欠压 | 250V | 切断输出 |

## 5. 接口设计

### 5.1 CAN报文设计

#### 接收报文（RX）
| 报文名 | CAN ID | 周期 | 字节 | 说明 |
|--------|--------|------|------|------|
| VCU_TorqueCmd | 0x101 | 10ms | 8 | 扭矩指令 |
| VCU_SpeedCmd | 0x102 | 10ms | 8 | 转速指令 |
| VCU_StateCtrl | 0x103 | 100ms | 8 | 状态控制 |

#### 发送报文（TX）
| 报文名 | CAN ID | 周期 | 字节 | 说明 |
|--------|--------|------|------|------|
| MCU_TorqueFb | 0x201 | 10ms | 8 | 扭矩反馈 |
| MCU_SpeedFb | 0x202 | 10ms | 8 | 转速反馈 |
| MCU_Status | 0x203 | 100ms | 8 | 状态信息 |
| MCU_Temperature | 0x204 | 100ms | 8 | 温度信息 |

### 5.2 诊断服务设计

| 服务ID | 功能 | 说明 |
|--------|------|------|
| 0x10 | 诊断会话控制 | 默认/编程/扩展会话 |
| 0x22 | 读数据标识符 | 读取参数 |
| 0x2E | 写数据标识符 | 写入参数 |
| 0x19 | 读DTC信息 | 故障码读取 |
| 0x14 | 清除DTC | 清除故障码 |
| 0x31 | 例程控制 | 执行测试例程 |

## 6. 状态机设计

### 6.1 主状态机

```
          ┌──────────────┐
          │     INIT     │
          └──────┬───────┘
                 │ 初始化完成
                 ▼
          ┌──────────────┐
          │   STANDBY    │<──────┐
          └──────┬───────┘       │
                 │ Enable请求    │ 故障恢复
                 ▼               │
          ┌──────────────┐       │
          │   PRECHARGE  │       │
          └──────┬───────┘       │
                 │ 预充完成      │
                 ▼               │
          ┌──────────────┐       │
          │     RUN      │───────┤ 正常运行
          └──────┬───────┘       │
                 │ 故障          │
                 ▼               │
          ┌──────────────┐       │
          │    FAULT     │───────┘
          └──────────────┘
```

### 6.2 状态定义

| 状态 | 说明 | 允许操作 |
|------|------|----------|
| INIT | 初始化状态 | 硬件自检 |
| STANDBY | 待机状态 | 低压工作，高压断开 |
| PRECHARGE | 预充状态 | 高压预充 |
| RUN | 运行状态 | 电机驱动 |
| FAULT | 故障状态 | 故障处理 |
