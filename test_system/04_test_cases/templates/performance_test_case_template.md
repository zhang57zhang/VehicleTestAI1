## 1 模板说明

本模板用于编写车载运动域/底盘域控制器的性能测试用例。性能测试用例验证被测系统在各种负载条件下的性能表现，包括响应时间测试、吞吐量测试、并发能力测试、负载测试、压力测试和稳定性测试。性能测试用例应明确性能指标、测量方法和通过标准。

### 3.1 示例测试用例

| 用例名称 | VCU CAN通信响应时间性能测试 |
| 用例类型 | 性能测试 |

**性能指标定义：**

| 字段 | 内容 |
|------|------|
| 指标名称 | CAN报文响应时间 |
| 指标定义 | 从VCU接收CAN报文，到VCU发出响应报文的时间间隔 |
| 目标值 | ≤50ms |
| 测量方法 | 使用CAN总线分析仪测量报文时间戳差值 |
| 测量点 | VCU CAN收发器的RX和TX引脚 |
| 采样频率 | 1MHz |

**负载模型定义：**

| 字段 | 内容 |
|------|------|

| 负载模式 | 恒定 |
| 预热时间 | 30秒 |

**测试步骤：**

| 步骤序号 | 操作描述 | 执行时间 | 验证内容 |

| 1 | 配置CAN总线分析仪，设置为监听模式 | 5分钟 | 设备连接正常 |
| 2 | 配置测试报文，设置为周期性发送（10ms周期） | 2分钟 | 报文参数正确 |
| 3 | 启动预热阶段，发送测试报文 | 30秒 | 系统状态稳定 |
| 4 | 启动正式测试，记录响应时间数据 | 300秒 | 数据采集正常 |
| 5 | 停止测试，保存测试数据 | 2分钟 | 数据完整 |

| 响应时间标准差 | 0ms | 10ms | 20% | 标准差≤10ms |
| 响应时间超限率 | 0% | 1% | 20% | 超限率≤1% |
