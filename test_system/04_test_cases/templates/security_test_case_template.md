## 1 模板说明

本模板用于编写车载运动域/底盘域控制器的安全测试用例。安全测试用例验证被测系统的安全功能和防护措施是否有效，包括功能安全测试、信息安全测试和操作安全测试。安全测试用例应参考ISO 26262功能安全标准，确保覆盖所有安全需求和安全机制。安全测试用例的编写应与功能安全工程师协作，确保测试场景符合功能安全要求。

## 3 填写示例

### 3.1 示例测试用例

**基本信息：**

| 字段 | 内容 |
|------|------|
| 用例编号 | TC-SAFE-ESP-001 |
| 用例名称 | ESP传感器信号丢失故障安全响应测试 |
| 用例类型 | 安全测试 |
| 测试类型 | 故障注入 |
| 所属模块 | 车身稳定系统ESP |
| 优先级 | 高 |
| 关联需求 | REQ-ESP-SAFE-001 |
| 关联场景 | SCN-SAFE-001 |
| ASIL等级 | D |
| 编写人 | 安全测试工程师 |
| 编写日期 | 2026-04-01 |

**安全需求定义：**

| 字段 | 内容 |
|------|------|
| 安全功能 | 轮速传感器信号丢失检测与安全状态控制 |
| 安全目标 | 当轮速传感器信号丢失时，系统应在100ms内检测到故障并进入安全状态 |
| 容错时间间隔FTTI | 300ms |
| 故障检测时间 | ≤100ms |
| 故障处理时间 | ≤50ms |
| 故障恢复时间 | ≤200ms |

**故障注入定义：**

| 字段 | 内容 |
|------|------|
| 故障类型 | 传感器故障 |
| 故障描述 | 左前轮轮速传感器信号丢失，传感器输出为0 |
| 注入方式 | 信号注入 |
| 注入参数 | 左前轮轮速信号线开路，等效阻抗无穷大 |
| 注入时机 | ESP系统处于正常运行状态，车速大于20km/h时注入 |
| 故障持续时间 | 持续注入直至测试完成 |

**测试步骤：**

| 步骤序号 | 操作描述 | 验证内容 | 预期结果 |
|----------|----------|----------|----------|
| 1 | 启动ESP系统，确认系统运行正常 | 系统状态 | 系统进入正常运行状态 |
| 2 | 驾驶车辆使车速达到30km/h | 车速信号 | 车速信号正常显示30km/h |
| 3 | 注入左前轮轮速传感器信号丢失故障 | 传感器信号 | 左前轮轮速信号变为0 |
| 4 | 监控ESP系统故障检测时间 | 故障检测时间 | 故障检测时间≤100ms |
| 5 | 观察ESP系统进入安全状态 | 安全状态 | ESP系统进入跛行回家模式 |
| 6 | 记录ESP系统故障处理时间 | 故障处理时间 | 故障处理时间≤50ms |
| 7 | 移除故障注入，观察系统恢复 | 系统恢复时间 | 系统在200ms内恢复正常或保持安全状态 |

**验证标准：**

| 验证项 | 验证内容 | 验证方法 | 通过标准 |
|----------|----------|----------|----------|
| 故障检测时间 | 从故障注入到故障检测的时间间隔 | 测量CAN报文时间戳 | ≤100ms |
| 故障处理时间 | 从故障检测到安全状态转换的时间 | 测量状态变化时间 | ≤50ms |
| 安全状态正确性 | ESP系统是否进入正确的安全状态 | 读取系统状态寄存器 | 进入跛行回家模式 |
| 故障指示灯 | 故障指示灯是否正确点亮 | 观察仪表盘 | 故障灯点亮 |
| 系统恢复 | 故障移除后系统是否正确恢复 | 观察系统状态 | 在规定时间内恢复 |

**测试执行信息：**

| 字段 | 内容 |
|------|------|
