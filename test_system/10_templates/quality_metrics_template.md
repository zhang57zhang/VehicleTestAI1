## 3 指标计算示例

### 3.1 用例执行率计算

```
用例执行率 = (实际执行用例数 / 计划执行用例数) × 100%

示例：
计划执行用例数 = 1000个
实际执行用例数 = 950个

用例执行率 = (950 / 1000) × 100% = 95%
```

### 3.2 缺陷发现率计算

```
缺陷发现率 = 缺陷总数 / 测试总工时

示例：
测试总工时 = 100人天
发现缺陷总数 = 85个

缺陷发现率 = 85 / 100 = 0.85个/人天
```

### 3.3 需求覆盖率计算

```
需求覆盖率 = (已验证需求数 / 总需求数) × 100%

示例：
总需求数 = 200条
已验证需求数 = 190条

需求覆盖率 = (190 / 200) × 100% = 95%
```
